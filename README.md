# Algoritme-Datastructuren Eindopdracht

Algoritme-Datastructuren eindopdracht van Rick Vinke en Victor Peters.

De applicatie maakt gebruik van Java en is getest met Java 19.

De data voor de collecties is bijgevoegd in het data.csv bestand.
Deze kan worden geimporteerd in de GUI.

### Collecties:
- Circular Buffer
- Linked List
- Tree Map

### Sorteer algoritmes:
- BubbleSort
- Balancing (alleen voor TreeMap)

### Zoek algoritmes:
- Sequential search
- Binary search
